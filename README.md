# ⚡ Smart Risk Detection System

🚀 AI-based system to predict risk level using heart rate and oxygen level.

---

## 🎯 About Project

This project was built during **Devcation Delhi 2026 (GDG IGDTUW x IIITD)** hackathon.

The hackathon environment inspired me to build something meaningful that can solve real-world problems using technology.

👉 This system predicts risk levels based on basic health parameters like Age, Heart Rate, and Oxygen Level.

---

## 🧠 How It Works

- User inputs:
  - Age
  - Heart Rate
  - Oxygen Level

- The backend (FastAPI) processes the data
- Returns risk level:

| Condition | Risk |
|----------|------|
| HR > 120 OR Oxygen < 90 | HIGH ⚠️ |
| HR > 100 | MEDIUM ⚡ |
| Else | LOW ✅ |
